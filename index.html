Welcome!

Welcome to the Episcopal High School Computer Science department website.

Here, the department will showcase its many projects and provide the source code for each.

Students develop iOS Applications using libraries used by companies like Facebook, such as React Native.
Students use Xcode and Swift to develop their own mobile applications and release them on the iOS App Store.
Students use Java and other languages to create useful programs with Graphical User Interfaces (GUIs).
